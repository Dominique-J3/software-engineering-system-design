# MaryMobile System Design

## Overview
This project focuses on designing a mobile-based class registration system to improve accessibility and usability.

## Problem
The existing system required desktop access and was slow during peak usage.

## Solution
Designed a mobile-friendly system that allows students to register for classes more efficiently.

## Key Components
- Use case diagrams
- Sequence diagrams
- Activity diagrams
- System requirements

## Features
- Mobile access to class registration
- Real-time updates on seat availability
- Improved user experience
